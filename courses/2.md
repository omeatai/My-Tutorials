# Node say Hello

### index.js:

```js
function sayHello() {
  return "Hello";
}

console.log(sayHello());
```

```jsbs
node index.js
```

### output:

```js
// nodejs % node index.js
//
// Hello
```

# Node with variables

### index.js:

```js
const mission = "learn";

if (mission === "learn") {
  console.log("Time to write some Node code!");
} else {
  console.log(`Is ${mission} really more fun?`);
}
```

```jsbs
node index.js
```

### output:

```js
// nodejs % node index.js
//
// Time to write some Node code!
```
